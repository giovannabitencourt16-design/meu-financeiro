# 💰 Meu Financeiro

Sistema de controle financeiro pessoal desenvolvido para ajudar usuários a organizar suas finanças de forma simples e intuitiva.

---

## 🚀 Funcionalidades

- 🔐 Login e cadastro de usuários
- 📊 Dashboard financeiro
- 💰 Cadastro de entradas
- 📄 Contas a pagar
- 🔄 Contas recorrentes automáticas
- 🔍 Pesquisa de contas
- 📅 Organização por vencimento
- 📈 Relatório mensal
- 🖨 Impressão e exportação para PDF
- 📱 Layout responsivo

---

## 🛠 Tecnologias

- HTML5
- CSS3
- JavaScript (ES6)
- Firebase Authentication
- Cloud Firestore

---

## 📂 Estrutura

```
css/
js/
pages/
firebase.rules
index.html
```

## 💡 Objetivo

Este projeto foi desenvolvido para praticar desenvolvimento Front-end utilizando Firebase como backend.
