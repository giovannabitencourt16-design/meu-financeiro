# Meu Financeiro v2

Inclui login, dashboard, entradas, contas, relatório e configurações.

Novidades:
- contas recorrentes automáticas
- forma de pagamento
- pesquisa de contas
- contas agrupadas por urgência
- atrasadas destacadas em vermelho
- contadores de pagas, pendentes e atrasadas no dashboard
- relatório mensal para imprimir ou salvar em PDF

Abra o index.html com Live Server. O Firebase já está configurado.
